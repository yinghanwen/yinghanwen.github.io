我的经历

前言

我写过很多人的传记，却唯独不了解自己。

我是Bob, Solitude, Beimo. 称号有很多，你可以叫我 Bob. 我是一名初三学生，兴趣爱好广泛，你可以和我聊心理学、马哲、独立游戏、二战史、Computer Science 等。

编程之路

我的父亲是一位程序员，受他影响我二年级开始学 Python 2. 七年级学习 Python3 和 Linux. 八年级学习 C 并且尝试了 Arch 和 Gentoo 等一些 Linux 系统，最后选择了 Ubuntu(笑)。同时我了解了游戏开发的相关知识，然而我的配置不够，我只能实践一些简单的东西，并学习有关的理论。

我却从始至终对 GUI 没兴趣，我认为浪费时间写 GUI 还不如提升一下核心代码。所以 QT, tk .net 等我一概不碰。

现在我正在学习 Macro Kernel 开发，还正在写一个基于 Nonebot2 和 Onebot V11 的机器人 Darlene

我的学历

我的小学是上海一个很出名的小学，甚至可以用贵族学校来形容。我通过了直升考，然而我的野心不止于此，我想要去上外一实，但是我差了 5 分没有去，却意外的被第三志愿录取，去一所“区重点”就读

游戏之旅

三岁的时候接触 Plants vs. Zombies，开启了游戏之旅

四岁时开始玩 PlayStation 2, 玩起了赛车游戏

五岁时玩起了 PlayStation 3, 体会了小小大星球

2018年在短视频平台接触刺激战场

五年级接触 Minecraft, 从此一发不可收拾，学起了红石和建筑。同时买了个 Xbox One S, 第一次玩到了光环，这个时候我才刚体会到什么是真正的游戏。同学给我介绍了第五人格，感觉很有意思，于是入坑。同时我开始玩起堡垒之夜，然而一直很菜，都在玩娱乐。同时接触了一个特别优秀的独立游戏：中国式家长。

六年级同学拉我入坑王者荣耀，花了几个月打上了钻石，然后去玩娱乐了。

七年级的时候，同学向我安利了 Phigros。从此，我对音游产生了浓厚兴趣，我在打完了三位魔王的 IN 难度后入坑了 Muse Dash, Arcaea, Maimaidx, Chunithm, Project Sekai. 然而我的底力不足，我还需要多多学习。班内同学向我推荐光遇，玩了一年后退坑。在 Steam 上买下了第一个游戏 Raft, 然后玩了 Escapist 2 并全部通关。接触了 L4D2，但是只在朋友家和游戏厅玩过。

八年级接触运动游戏：Fifa, 2k, 实况足球等。和朋友一起入坑了原神，由于网课时间充裕，一直在肝原神。后续感觉只有剧情有意思，一直肝游戏有点像上班，于是淡坑。之后入坑 Apex Legends.

九年级接触策略类游戏：钢铁雄心4，由于我正好喜欢二战史，所以我深深爱上了这个游戏。接触了 galgame: ddlc, 东北之夏，意外得知自己可以写出很恐怖的剧情（笑）

艺术造诣

小学时学习钢琴，考级考到了英皇二级，没有继续下去。美术课模仿毕加索、梵高等人的艺术风格并画了几幅画。

初中接触到 Hardcore tano c, 喜欢 tpz 和 USAO 的曲风，现在正在学习人声采样切片和鼓组，之后会学习乐理。

学科竞赛

我在三年级时曾学过数学竞赛，然而天没亮就起床，然后坐两个小时地铁去上课，太苦了，我没有坚持下去。八年级下时，我妈给我报了个物理竞赛班，力学和热学对我来说很简单。我自以为很有天赋。然而电学给了我沉重的一击，学校的物理成绩也下降了，遂决定退出物理竞赛。OI 我并没有太大兴趣，我认为程序隔了算法，就只能看到他的朦胧美。虽然我初中化学学的很好，但我对化学竞赛不感兴趣。我认为生物竞赛还是蛮有意思的。21世纪是生物的世纪。我喜欢这种体系不完善，而且用不到多少数学，知识面广的学科。

我的梦想

初中是一个理想主义的年纪。我想去上海交通大学就读生物工程专业。然后去剑桥大学或斯坦福大学读硕士，如果能拿到 CS 和生物工程双学位那就更好了。之后我会在国外读研，然后回上海工作。我从小的梦想就是当科学家，因此我以后也会走上科研的道路。

政治理想

我讨厌权威，憎恶权力。身边的人认为我不现实，给我贴标签，认为我过于远视主义。而我认为英特纳雄耐尔必将实现。世界共产主义运动的起点会是第三世界国家，然后会向欧美资本主义大本营发展。生产力会变得极度发达，政府、国家会消失。寰球插满了赤旗，人类开始探索浩瀚的宇宙。

尾声

青春就是试错的机会，我正在一条崎岖的路上坎坷行走，不断试错。

无论他人怎样攻击我，精神变态也好，利己主义也罢，我不应当被定义，我就是我。这些只是我青春路上的绊脚石。

和朋友倾诉，大抵也会被当作他们的谈资罢。这里并不是安那其社会，大部分人是不可信任的。因此，我建立了这个博客，也许是为了记录自己的遐想，也许只是为了给我无处安放的分享欲一个信箱罢了。

Bob Ying

An artist
